## ğŸ§© Lessons Overview

### Lesson 1: HTML Text Elements
- **Learning**: Introduction to HTML text formatting elements
- **Topics covered**: 
  - HTML headings (h1-h6)
  - Paragraphs (p)
  - Line breaks (br)
  - Text formatting (strong, b, em, i, u)
  - Combining multiple formatting elements

### Lesson 2: CSS Learning Guide
- **Learning**: CSS concepts and techniques
- **Topics covered**:
  - Inline CSS vs Internal CSS
  - CSS Box Model (content, padding, border, margin)
  - CSS Selectors (element, class, ID, descendant, child, etc.)
  - CSS Specificity

### Lesson 3: Styling Images in CSS
- **Learning**: Advanced image styling techniques
- **Topics covered**:
  - Display Flex for image arrangement
  - Border and Border-radius properties
  - Box-shadow effects
  - Opacity control
  - Background images
  - CSS filters and hover effects
