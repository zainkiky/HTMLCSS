Styling Images in CSS

Learn how to enhance and customize images with CSS properties

1. Display Flex

display: flex; creates a flexible container that allows you to easily align and distribute elements, including images.

Key properties when using flex:

  • justify-content: Aligns items horizontally
  • align-items: Aligns items vertically
  • flex-direction: Sets the direction of flex items

2. Border

border adds a visible edge around an image. You can customize the border with:

  • Border width (e.g., 2px, thick)
  • Border style (e.g., solid, dashed, dotted)
  • Border color (e.g., red, #FF0000)

No Border

3. Border-radius

border-radius rounds the corners of an image. You can specify:

  • A single value for all corners (e.g., 10px)
  • Different values for horizontal and vertical radii (e.g., 10px / 20px)
  • Individual values for each corner (e.g., 10px 5px 10px 5px)

Setting border-radius to 50% creates a circular image.

4. Box-shadow

box-shadow adds shadow effects around an image. It accepts multiple values:

  • Horizontal offset (e.g., 5px)
  • Vertical offset (e.g., 10px)
  • Blur radius (e.g., 8px)
  • Spread radius (e.g., 2px)
  • Color (e.g., rgba(0,0,0,0.3))

5. Opacity

opacity controls the transparency of an image. Values range from 0 (completely transparent) to 1 (completely opaque).

6. Image Background

Instead of using an <img> tag, you can use CSS background properties to place images:

  • background-image: Sets the image
  • background-size: Controls how the image scales
  • background-position: Sets the position of the image
  • background-repeat: Defines if and how the image repeats
