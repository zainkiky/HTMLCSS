Introduction to CSS

This page demonstrates CSS concepts: Inline CSS, Internal CSS, Box Model, and CSS Selectors

1. Inline CSS

Documentation: Inline CSS is applied directly to an HTML element using the style attribute. It has the highest specificity and overrides other CSS rules.

Inline CSS Examples:

This paragraph has inline CSS - red color and 18px font size

This paragraph has inline CSS - yellow background, blue border, and padding

This span has inline CSS styling

HTML Code for Inline CSS:

<p style="color: red; font-size: 18px;">This paragraph has inline CSS - red color and 18px font size</p>
<p style="background-color: yellow; padding: 10px; border: 2px solid blue;">This paragraph has inline CSS - yellow background, blue border, and padding</p>
<span style="font-weight: bold; text-decoration: underline;">This span has inline CSS styling</span>

2. Internal CSS

Documentation: Internal CSS is defined within the <style> tag in the HTML document's <head> section. It applies to the entire HTML document.

Internal CSS Examples:

This paragraph is styled by the internal CSS in the <head> section.

This paragraph has a class that's styled by internal CSS.

This div is styled by an ID selector defined in internal CSS.

HTML Code for Internal CSS:

<head>
    <style>
        p {
            color: #2c3e50;
            font-family: Arial, sans-serif;
        }
        .example-internal {
            background-color: #dfe6e9;
            padding: 10px;
        }
        #internal-example {
            background-color: #fd79a8;
            color: white;
            padding: 10px;
            border-radius: 5px;
        }
    </style>
</head>

3. CSS Box Model

Documentation: The CSS box model describes how every HTML element is represented as a rectangular box with four areas:

  • Content: The actual content of the element
  • Padding: Space between the content and the border
  • Border: The border that goes around the padding and content
  • Margin: Space outside the border

Box Model Visualization:

Content Area

Box Model Properties Table:

Property Description Example
Content The actual content of the element (text, images, etc.) The text "Content Area" in the blue box
Padding Space between the content and the border (15px in example) The space around the content
Border The border around the padding (5px solid border in example) The blue border (5px solid)
Margin Space outside the border (20px in example) The space between the box and surrounding elements

HTML Code for Box Model:

<div style="background-color: #dfe6e9; padding: 20px;">
    <div style="background-color: #74b9ff; 
                 width: 200px; height: 100px;
                 margin: 20px; 
                 padding: 15px; 
                 border: 5px solid #0984e3;
                 color: white;
                 text-align: center;
                 line-height: 100px;">
        Content Area
    </div>
</div>

Complete CSS Selectors Code Example:

/* Element Selector */
p {
    color: #2c3e50;
}

/* Class Selector */
.class-selector-example {
    background-color: #a29bfe;
    padding: 8px;
    border-radius: 4px;
}

/* ID Selector */
#id-selector-example {
    background-color: #fd79a8;
    color: white;
    padding: 8px;
    border-radius: 4px;
}

/* Universal Selector */
* {
    box-sizing: border-box;
}

/* Descendant Selector */
.documentation p {
    font-style: italic;
    color: #e17055;
}

/* Child Selector */
.child-example > span {
    background-color: #74b9ff;
    color: white;
    padding: 3px 6px;
    border-radius: 3px;
}

/* Adjacent Sibling Selector */
.example p + p {
    background-color: #ffeaa7;
    padding: 5px;
    border-left: 3px solid #e17055;
}

/* Attribute Selector */
a[href*="example"] {
    color: #00b894;
    font-weight: bold;
}

/* Pseudo-class Selector */
.example p:first-child {
    font-weight: bold;
    background-color: #fab1a0;
}

/* Pseudo-element Selector */
.example p::first-letter {
    font-size: 1.8em;
    color: #e17055;
    font-weight: bold;
}

/* Grouping Selector */
code, pre {
    background-color: #2d3436;
    color: #fff;
    padding: 2px 5px;
    border-radius: 3px;
}

Selector Specificity Comparison:

Selector Type Specificity Value Example
Inline CSS 1000 style="color: red;"
ID Selector 0100 #example
Class Selector 0010 .example
Element Selector 0001 p, div, span
Universal Selector 0000 *

5. Summary

Key Takeaways:

  • Inline CSS: Applied with style attribute, has highest specificity
  • Internal CSS: Defined in <style> tag in <head>, applies to entire document
  • CSS Box Model: Content + Padding + Border + Margin defines every element's box
  • CSS Selectors: Various ways to select elements including element, class, ID, attribute, pseudo-class, and more
  • Specificity: Inline > ID > Class > Element > Universal, determines which styles are applied
