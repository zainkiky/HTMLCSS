HTML Text Elements - Educational Guide

This page demonstrates the basic HTML text formatting elements used for educational purposes.

1. Headings

Documentation: HTML provides 6 levels of headings using <h1> to <h6> tags. <h1> is the most important (largest) and <h6> is the least important (smallest).

Heading Examples:

This is an H1 Heading

This is an H2 Heading

This is an H3 Heading

This is an H4 Heading

This is an H5 Heading
This is an H6 Heading

HTML Code for Headings:

<h1>This is an H1 Heading</h1>
<h2>This is an H2 Heading</h2>
<h3>This is an H3 Heading</h3>
<h4>This is an H4 Heading</h4>
<h5>This is an H5 Heading</h5>
<h6>This is an H6 Heading</h6>

2. Paragraph

Documentation: The <p> tag defines a paragraph. Browsers automatically add some space before and after each paragraph.

Paragraph Example:

This is a paragraph element. Paragraphs are used to group related content together in a block of text. They help organize content for better readability.

This is another paragraph to show how multiple paragraphs work. Each paragraph creates a new block of text with space before and after it.

HTML Code for Paragraph:

<p>This is a paragraph element. Paragraphs are used to group related content together in a block of text.</p>
<p>This is another paragraph to show how multiple paragraphs work.</p>

3. Line Break

Documentation: The <br> tag inserts a line break in text. It's an empty tag without a closing tag. It's useful when you want a new line without starting a new paragraph.

Line Break Example:

This is the first line.
This is the second line after a line break.
This is the third line with another break.

HTML Code for Line Break:

<p>This is the first line.<br>
This is the second line after a line break.<br>
This is the third line with another break.</p>

4. Bold Text

Documentation: The <strong> tag is used to define important text and is typically displayed in bold. The <b> tag also makes text bold but doesn't convey importance semantically.

Bold Text Examples:

This is normal text with strong (bold) text in the middle.

This is normal text with bold (b tag) text in the middle.

HTML Code for Bold Text:

<p>This is normal text with <strong>strong (bold)</strong> text in the middle.</p>
<p>This is normal text with <b>bold (b tag)</b> text in the middle.</p>

5. Italic Text

Documentation: The <em> tag is used to define emphasized text and is typically displayed in italics. The <i> tag also makes text italic but doesn't convey emphasis semantically.

Italic Text Examples:

This is normal text with emphasized (italic) text in the middle.

This is normal text with italic (i tag) text in the middle.

HTML Code for Italic Text:

<p>This is normal text with <em>emphasized (italic)</em> text in the middle.</p>
<p>This is normal text with <i>italic (i tag)</i> text in the middle.</p>

6. Underline Text

Documentation: The <u> tag is used to underline text. This tag is often used for proper names in Chinese text, or to label certain parts of text as having a specific meaning or usage.

Underline Text Examples:

This is normal text with underlined text in the middle.

Another example with underline in the text.

HTML Code for Underline Text:

<p>This is normal text with <u>underlined</u> text in the middle.</p>
<p>Another example with <u>underline</u> in the text.</p>

7. Combined Formatting Examples

Documentation: You can combine multiple formatting elements within the same text to create rich formatting.

Combining Elements:

This paragraph contains bold text, italic text, underlined text, and bold and italic text.

Here is bold and underlined text as well as italic and underlined text.

HTML Code for Combined Formatting:

<p>This paragraph contains <strong>bold text</strong>, <em>italic text</em>, <u>underlined text</u>, and <strong><em>bold and italic text</em></strong>.</p>
<p>Here is <strong><u>bold and underlined text</u></strong> as well as <em><u>italic and underlined text</u></em>.</p>

8. Summary

Key Takeaways:

  • Use <h1> to <h6> tags for headings (h1 being most important)
  • Use <p> tag for paragraphs
  • Use <br> tag for line breaks within text
  • Use <strong> for important/bold text (semantic meaning)
  • Use <em> for emphasized/italic text (semantic meaning)
  • Use <u> tag for underlined text
  • Combine formatting elements for enhanced text styling
